About project
The HRMS project is built using a modern tech stack, including Angular for the front end, Python and Django for the back end, and dbsqlite3 for the database. This combination ensures a system that can adapt to the evolving needs of an organization.
I recently completed the Python Fullstack Development course at IKUB Info Academy, which equipped me with the skills to build full-stack applications using modern development frameworks.
